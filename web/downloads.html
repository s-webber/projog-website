Console Application

If you want to use Projog as a command-line interface to Prolog then download and unzip projog-@PROJOG_VERSION@.zip.

Maven Artifact

If you want to integrate Projog into a Java project then, if the project is configured using Maven, you can add a dependency on Projog by adding the following to your project's pom.xml:

<dependency>
   <groupId>org.projog</groupId>
   <artifactId>projog-core</artifactId>
   <version>@PROJOG_VERSION@</version>
</dependency>

The projog-examples GitHub project contains examples of integrating Projog with Java.
